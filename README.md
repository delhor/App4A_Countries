# PRESENTATION

Application rélisée par Daoud ELHOR, un apprenti ingénieur en 4A à l'ESIEA Paris.
L'application démarre avec un magnifique écran contenant le magnifique drapeau de la Tunisie ainsi qu'une barre de progression.
Cette application affiche une liste d'élément provenant d'une API-REST sur le 2ème écran. Et elle affiche les détails de chaque élémentde la liste dans un 3ème écran.
La liste d'élément est composé de tous les noms de pays du monde entier ainsi que leur drapeau. 
Dans les détails, on retrouve la région(Europe,Asie...), la capitale et la population.

L'application utilise l'API-REST : https://restcountries.eu/rest/v2/all

# PREREQUIS

- Installation d'Android Studio
- Connexion Internet

# CONSIGNES RESPECTEES

- Appel REST
- Ecran : 3 activités (Démarrage + Liste + Détails)
- Affichage d'une liste dans un Recycler View
- Affichage du détail des items de la liste
- Achitecture MVC
- Singleton
- Fragments
- Gitflow
